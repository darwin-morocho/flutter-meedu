## Features
- A State Managment for simple and complex states.
- Navigate without BuildContext.
- Simple dependecy injection.
- Reactive programming.

## Getting Started

Add the following to your `pubspec.yaml`

```yaml
environment:
  sdk: ">=2.12.0 <3.0.0"
  flutter: ">=2.0.0"

dependencies:
  flutter_meedu: "^3.0.8"
```
